# python-pip
python Pip Repository



# 更新pip 到最新的版本：

```
python -m pip install --upgrade pip
```

## 配置文件使用

`Linux`下，修改~/.pip/pip.conf（没有就创建一个文件及文件夹，文件夹要加“.”，表示隐藏文件夹）
`Windows`下，直接在user目录中创建一个pip目录，如C:\Users\xx\pip，新建文件pip.ini。
